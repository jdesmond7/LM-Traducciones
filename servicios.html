¿Qué tipo de documentos puedes enviar para revisión?

Puedes enviar documentos escaneados o digitales en formatos comunes como PDF, Word o imagen, siempre que sean legibles y estén completos. Antes de emitir una cotización final, se revisa el material para confirmar extensión, tipo de documento y nivel de especialización requerido.

  • PDF
  • DOC / DOCX
  • JPG / PNG

Es importante que el documento se encuentre completo y sea claramente legible para realizar una evaluación adecuada.

¿No estás seguro de qué tipo de traducción necesitas?

Si tienes dudas sobre la categoría de tu documento, puedes enviar tu archivo o describirlo en el formulario de contacto. Con gusto revisaré tu caso y te indicaré el tipo de servicio adecuado, así como el procedimiento a seguir.
